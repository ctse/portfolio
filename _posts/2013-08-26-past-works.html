University of Technology, Sydney

Digital Multimedia

Group (of four students) Multimedia project spanning an entire semester.

  • Denko Anime@UTS Site

    Role: Front-end Developer and Graphics Design
    This site was created for Digital Multimedia at UTS as an alternate 'gamified' site for Anime@UTS, based on Foursquare. The aim was to allow users to be more engaged with the club and their fellow members and to gain rewards for attendance.

Digital Media Technologies

Individual website creation using HTML, CSS and PHP spanning an entire semester.

  • LearnJapanese Documentation Site

    This site was created for the purposes of site documentation. The aim was to allow the different deliverables to be loaded using jQuery, with each deliverable looking like a paper, hardcopy document on a wooden table for the user to read to improve user experience.

  • LearnJapanese with PenguinCoffee

    This site was created for the subject Digital Media Technologies, at UTS. The aim was to help those wanting to learn the Japanese writing system of Hiragana, by using more visual means to link images with characters. Additionally, users could save their progress and potentially interact.

Interface Design

Group assignment of a high fidelity prototype of a futuristic system.

  • Heads-up Display Hi-fi Prototype

    Role: Front-end and JavaScript/jQuery Developer
    This site was created as a high fidelity prototype for Interface Design at UTS for a futuristic system to be used onboard a spaceship. Multiple options such as allowing the user to be awoken from sleep by music were simulated with jQuery.

Digital and Multiplatform Storytelling

Group assignment of creating a storytelling project based on a proposal completed as a previous assessment. Project I was involved in was for an iPad app that teaches people about authentic Chinese cuisine, with social media connections.

  • Hua's Food Map

    Role: Graphics design, page design, programming
    This application was created to a pre-approved proposal for an application called 'Hua's Food Map' which taught users about authentic Chinese cuisine based on four food factions. History is provided as well as recipes.
